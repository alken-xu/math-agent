## 逐题审核

### 第7题
- **solver答案**：m = 11/5 或 m = 9/7
- **验证结论**：⚠️ 答案正确，方法超纲
- **正确答案**：m = 11/5 或 m = 9/7（答案无误）
- **说明**：
  - 题目：M(m+1, 3m-5) 到 x 轴距离是到 y 轴距离的一半，即 |3m-5| = (1/2)|m+1|。
  - Solver 将绝对值方程两边平方转化为一元二次方程，再使用求根公式求解。**求根公式属于九年级内容，在七年级不应使用**。
  - 七年级正确方法：分情况去绝对值，化为两个一元一次方程：
    - 2(3m-5) = m+1 → m = 11/5
    - 2(3m-5) = -(m+1) → m = 9/7
  - 两种方法答案一致，最终答案正确。

- **正确答案**：共四种情况
  - A(0,0)，B(2,0)，C(2,2)，D(0,2)
  - A(0,0)，B(-2,0)，C(-2,2)，D(0,2)
  - A(0,0)，B(2,0)，C(2,-2)，D(0,-2)
  - A(0,0)，B(-2,0)，C(-2,-2)，D(0,-2)

  - 若题目图示中给出了方向（如图示 B 在 x 轴正半轴、D 在 y 轴正半轴），则只需 A(0,0)、B(2,0)、C(2,2)、D(0,2) 一种即可。图片中第18题附有示意图，但第8题无图，故应穷举所有情况。

### 第9题
- **solver答案**：C(2, 2)
- **验证结论**：✅ 正确
- **说明**：
  - 平行四边形 ABCD，AB∥x轴，CD∥x轴，AB 长度 = |1-(-3)| = 4，方向为向右。
  - D(-2,2) 沿与 AB 同向移动 4 个单位：C = (-2+4, 2) = (2,2)。
  - 验证：C(2,2) 与 D(-2,2) 纵坐标相同，CD∥x轴 ✓。
  - Solver 使用了"向量"表述，但运算逻辑等价于七年级坐标平移，答案正确。

### 变1题
- **solver答案**：D(-6, 2)
- **验证结论**：✅ 正确
- **说明**：
  - A(-3,0)，B(1,0)，C(-2,2)，利用对角线互相平分：
  - AC 中点 = ((-3-2)/2, (0+2)/2) = (-5/2, 1)
  - BD 中点也等于 (-5/2, 1)，D = (2×(-5/2)-1, 2×1-0) = (-6, 2) ✓
  - 验证：AB = (4,0)，DC = (-2-(-6), 2-2) = (4,0)，AB = DC ✓。
