## 第7题

**题目**：M(m+1, 3m-5) 到 x 轴距离是它到 y 轴距离的一半，求 m。

**知识点**：平面直角坐标系中点到坐标轴的距离

**解题过程**：
- 点到 x 轴的距离 = |纵坐标| = |3m-5|
- 点到 y 轴的距离 = |横坐标| = |m+1|

由题意：|3m-5| = (1/2)|m+1|

即 2|3m-5| = |m+1|

两边平方：4(3m-5)² = (m+1)²

展开：4(9m²-30m+25) = m²+2m+1

36m²-120m+100 = m²+2m+1

35m²-122m+99 = 0

用求根公式：Δ = 122²-4×35×99 = 14884-13860 = 1024，√Δ = 32

m = (122±32)/70

m₁ = 154/70 = **11/5**
m₂ = 90/70 = **9/7**

**自查**：
- m = 11/5：|3×11/5-5| = 8/5，|11/5+1| = 16/5，8/5 = (1/2)×16/5 ✓
- m = 9/7：|3×9/7-5| = 8/7，|9/7+1| = 16/7，8/7 = (1/2)×16/7 ✓

**答案**：m = 11/5 或 m = 9/7

**答案**：A(0,0)，B(2,0)，C(2,2)，D(0,2)

## 第9题

**题目**：平行四边形 ABCD 中，AB 与 x 轴重合，CD∥x 轴，若 A(-3,0)，B(1,0)，D(-2,2)，求 C 点坐标。

**知识点**：平行四边形性质（对边平行且相等），坐标运算

**解题过程**：

平行四边形 ABCD 中，向量 AB = 向量 DC（对边平行且相等）。

向量 AB = B - A = (1-(-3), 0-0) = (4, 0)

C = D + 向量 DC = D + (4,0) = (-2+4, 2+0) = (2, 2)

**自查**：CD∥x 轴，C(2,2) 和 D(-2,2) 的纵坐标均为 2，确实平行于 x 轴。✓

**答案**：C(2, 2)

## 变1题

**题目**：若 A(-3,0)，B(1,0)，C(-2,2)，求 D 点坐标（平行四边形 ABCD）。

**知识点**：平行四边形性质（对角线互相平分），坐标运算

**解题过程**：

对角线 AC 与 BD 互相平分，设交点为 M。

M = 中点(A,C) = ((-3+(-2))/2, (0+2)/2) = (-5/2, 1)

D = 2M - B = (2×(-5/2)-1, 2×1-0) = (-5-1, 2) = (-6, 2)

**自查**：向量 AB = (4,0)，向量 DC = (-2-(-6), 2-2) = (4,0)，相等。✓

**答案**：D(-6, 2)

## 变2题

**题目**：平行四边形三个顶点坐标分别为(-3,0)，(1,0)，(-2,-2)，求另一个顶点坐标。

**知识点**：平行四边形性质，对角线互相平分，分情况讨论

**解题过程**：

设三个已知顶点为 P₁(-3,0)，P₂(1,0)，P₃(-2,-2)，第四个顶点为 P₄。

由于不知道哪两个点是对角顶点，分三种情况：

**情况一**：P₁ 与 P₃ 是对角顶点
中点 = ((-3-2)/2, (0-2)/2) = (-5/2, -1)
P₄：(1+x)/2 = -5/2 → x = -6；(0+y)/2 = -1 → y = -2
P₄ = **(-6, -2)**

**情况二**：P₁ 与 P₂ 是对角顶点
中点 = ((-3+1)/2, (0+0)/2) = (-1, 0)
P₄：(-2+x)/2 = -1 → x = 0；(-2+y)/2 = 0 → y = 2
P₄ = **(0, 2)**

**情况三**：P₂ 与 P₃ 是对角顶点
中点 = ((1-2)/2, (0-2)/2) = (-1/2, -1)
P₄：(-3+x)/2 = -1/2 → x = 2；(0+y)/2 = -1 → y = -2
P₄ = **(2, -2)**

**答案**：第四个顶点坐标为 (-6,-2) 或 (0,2) 或 (2,-2)（三种情况均需列出）

## 第12题

**题目**：已知 M(3a-8, a-1)，分别根据下列条件求 M 点的坐标：
① M 在 y 轴上；
② M 在二、四象限角平分线上（OM 平分∠xOy）；
③ M 在第二象限且 a 为整数；
④ N(3,-6)，MN∥x 轴。

**知识点**：坐标轴上点的坐标特征，象限角平分线，平行于坐标轴

**解题过程**：

**① M 在 y 轴上**（横坐标为 0）：
3a-8 = 0 → a = 8/3
纵坐标 = 8/3 - 1 = 5/3
**M(0, 5/3)**

**② M 在二、四象限角平分线上**（该直线满足 x + y = 0）：
(3a-8) + (a-1) = 0
4a - 9 = 0 → a = 9/4
横坐标 = 3×9/4-8 = 27/4-32/4 = -5/4
纵坐标 = 9/4-1 = 5/4
验证：-5/4 + 5/4 = 0 ✓，且 -5/4 < 0，5/4 > 0，在第二象限 ✓
**M(-5/4, 5/4)**

**③ M 在第二象限且 a 为整数**：
第二象限要求：3a-8 < 0（横坐标 < 0）且 a-1 > 0（纵坐标 > 0）
→ a < 8/3 ≈ 2.67，且 a > 1
a 为整数，故 a = 2
横坐标 = 3×2-8 = -2，纵坐标 = 2-1 = 1
**M(-2, 1)**

**④ MN∥x 轴**（两点纵坐标相等）：
a-1 = -6 → a = -5
横坐标 = 3×(-5)-8 = -23
**M(-23, -6)**

**答案**：① M(0, 5/3)；② M(-5/4, 5/4)；③ M(-2, 1)；④ M(-23, -6)

## 第13题

**题目**：已知 M(|m|, n) 且 mn≠0，M 在第几象限？

**知识点**：绝对值性质，象限判断

**解题过程**：

mn≠0 说明 m≠0 且 n≠0。

横坐标 |m| > 0（绝对值且 m≠0，恒为正）

纵坐标 n 的符号不确定：
- 若 n > 0：横坐标 > 0，纵坐标 > 0 → 第一象限
- 若 n < 0：横坐标 > 0，纵坐标 < 0 → 第四象限

**答案**：M 在第一象限或第四象限

## 第14题

**题目**：若 M(a, a-b) 在第四象限，那么 Q(b, -a) 在第几象限？

**知识点**：象限条件，不等式推断

**解题过程**：

M 在第四象限：
- a > 0（横坐标 > 0）
- a-b < 0，即 b > a > 0，故 b > 0（纵坐标 < 0）

对于 Q(b, -a)：
- 横坐标 b > 0
- 纵坐标 -a < 0（因 a > 0）

**答案**：Q 在第四象限

## 第16题

**题目**：若 P(4x-5, 3-x) 在第四象限，化简 |5-4x| - |x-3|

**知识点**：象限条件，绝对值化简

**解题过程**：

P 在第四象限的条件：
- 4x-5 > 0 → x > 5/4
- 3-x < 0 → x > 3

综合：x > 3

在 x > 3 时：
- 5-4x < 0（因 4x > 12 > 5），故 |5-4x| = -(5-4x) = 4x-5
- x-3 > 0，故 |x-3| = x-3

化简：
|5-4x| - |x-3| = (4x-5) - (x-3) = 4x-5-x+3 = **3x-2**

**答案**：3x-2

## 第17题

**题目**：已知 P(2x+3, 3-y)，Q(y+4, x-5)，若 P、Q：① 关于 x 轴对称；② 关于 y 轴对称；③ 关于原点对称，分别求出相应的 x，y 的值。

**知识点**：关于坐标轴和原点对称的坐标关系

**解题过程**：

**① 关于 x 轴对称**：横坐标相等，纵坐标互为相反数
```
2x+3 = y+4    ...(i)
3-y = -(x-5)  ...(ii)
```
由(ii)：3-y = -x+5 → x-y = 2 → x = y+2
代入(i)：2(y+2)+3 = y+4 → 2y+7 = y+4 → y = -3，x = -1

验证：P(2×(-1)+3, 3-(-3)) = P(1,6)；Q(-3+4,-1-5) = Q(1,-6)；关于 x 轴对称 ✓

**答：x = -1，y = -3**

**② 关于 y 轴对称**：横坐标互为相反数，纵坐标相等
```
2x+3 = -(y+4)  ...(iii)
3-y = x-5      ...(iv)
```
由(iv)：x = 8-y
代入(iii)：2(8-y)+3 = -(y+4) → 16-2y+3 = -y-4 → -y = -23 → y = 23，x = -15

验证：P(2×(-15)+3, 3-23) = P(-27,-20)；Q(23+4,-15-5) = Q(27,-20)；关于 y 轴对称 ✓

**答：x = -15，y = 23**

**③ 关于原点对称**：横坐标互为相反数，纵坐标互为相反数
```
2x+3 = -(y+4)  ...(v)
3-y = -(x-5)   ...(vi)
```
由(vi)：3-y = -x+5 → x = y+2
代入(v)：2(y+2)+3 = -(y+4) → 2y+7 = -y-4 → 3y = -11 → y = -11/3，x = -5/3

验证：P(2×(-5/3)+3, 3-(-11/3)) = P(-1/3, 20/3)；Q(-11/3+4,-5/3-5) = Q(1/3,-20/3)；关于原点对称 ✓

**答：x = -5/3，y = -11/3**

## 第18题

**题目**：在平面直角坐标系中，AB 是过(1,0)且垂直于 x 轴的平面镜，则点 P(3,2) 在 AB 中的像 P' 的坐标为___。

**知识点**：关于竖直直线（x = 常数）的对称点坐标

**解题过程**：

AB 是直线 x = 1（过(1,0)且垂直于 x 轴）。

点 P(3,2) 关于直线 x = 1 的对称点 P'：
- 纵坐标不变：y' = 2
- 横坐标：P 的横坐标 3 到直线 x=1 的距离为 3-1 = 2，对称后 x' = 1-2 = -1

**自查**：P(3,2) 和 P'(-1,2) 的中点横坐标 = (3-1)/2 = 1，在直线 x=1 上 ✓

**答案**：P'(-1, 2)
